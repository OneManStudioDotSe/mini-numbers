How It Works

Mini Numbers is a self-hosted web analytics platform. You deploy it on your own server, configure it through a browser-based wizard, and add a tiny script to your website. Here's how it all fits together.

1. Install

Mini Numbers runs as a Docker container or a standalone Java application. The fastest way to get started is with Docker Compose:

Terminal 
git clone https://github.com/OneManStudioDotSe/mini-numbers.git
cd mini-numbers
docker compose up -d

Prerequisites

  • Docker Compose (recommended) — any modern version
  • Or Docker standalone
  • Or JDK 21+ for building from source

The Docker image is multi-platform (amd64, arm64) and weighs approximately 200MB.

2. Configure

When you first visit your Mini Numbers instance, the setup wizard guides you through configuration in 5 steps:

  1. Security — Set admin username, password, and server salt
  2. Database — Choose SQLite (default, zero-config) or PostgreSQL
  3. Server — Configure port, CORS origins, and development mode
  4. Advanced — GeoIP database path and rate limiting
  5. Review — Confirm settings and launch

The wizard saves a .env file and initializes all services — no server restart required. This is Mini Numbers' zero-restart architecture in action.

3. Add the Tracker

Add one script tag to your website's HTML. That's the entire integration:

your-website.html 
<script
  async
  src="https://your-analytics.com/tracker/tracker.js"
  data-project-key="YOUR_API_KEY"
></script>

Tracker Features

  • 1.3KB minified — smaller than most favicon images
  • SPA support — automatically patches History API for single-page apps
  • Heartbeat tracking — measures actual time-on-page (visibility-aware)
  • Custom events — track any interaction with MiniNumbers.track("event")
  • No cookies — uses sessionStorage for session management

Configuration Options

Attribute Default Description
data-project-key Required Your project's API key
data-api-endpoint Script origin Custom API endpoint URL
data-heartbeat-interval 30000 Heartbeat interval in ms
data-disable-spa false Disable SPA tracking

4. Analyze

Visit /admin-panel on your Mini Numbers instance to access the dashboard. You'll see:

  • Page views, unique visitors, and bounce rate with period comparison
  • Top pages, referrers, browsers, and countries
  • Activity heatmaps and contribution calendars
  • Real-time visitor feed with live location tracking
  • Conversion goals, funnels, and user segments

Privacy Architecture

Mini Numbers never stores IP addresses. Instead, it creates a one-way hash:

Privacy hash 
visitorHash = SHA256(ip + userAgent + projectId + serverSalt + rotationBucket)

The hash rotates on a configurable schedule (1 to 8,760 hours). After rotation, old visitors appear as new — preventing long-term tracking. Three privacy modes let you control exactly how much data is collected.

For full details, read the Privacy Architecture Documentation.

Privacy is not a feature. It's the architecture.

Your visitors' data never leaves your server. No third-party services, no external calls, no tracking across sites.

visitorHash = SHA256(
  ip + userAgent + projectId
  + serverSalt + rotationBucket
)

No Cookies

Uses sessionStorage for session tracking. No consent banner needed — ever.

No IP Storage

IP addresses exist only in RAM during request processing. Never written to disk.

Configurable Hash Rotation

Rotate visitor hashes every 1 to 8,760 hours. A feature unique to Mini Numbers.

Up and running in minutes

Choose your preferred deployment method. All roads lead to a working analytics server.

Terminal 
# Clone the repository
git clone https://github.com/OneManStudioDotSe/mini-numbers.git
cd mini-numbers

# Start with Docker Compose
docker compose up -d

# Visit https://analytics.example.com for the setup wizard
Terminal 
# Build the image
docker build -t mini-numbers .

# Run the container
docker run -p 8080:8080 mini-numbers

# Visit https://analytics.example.com for the setup wizard
Terminal 
# Build the fat JAR (requires JDK 21)
./gradlew buildFatJar

# Run the application
java -jar build/libs/mini-numbers-all.jar

# Visit https://analytics.example.com for the setup wizard
